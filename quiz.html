To calculate the area of a circle, use the formula:

\[ A = \pi r^2 \]

Here's a Python function to calculate it:

import math

def calculate_circle_area(radius):
    """Calculate the area of a circle given its radius"""
    return math.pi * (radius ** 2)

# Example usage
print(calculate_circle_area(5))  # Output: 78.53981633974483

  • Uses Python's built-in math module for π
  • Follows PEP 8 style guidelines
  • Includes a docstring for documentation
